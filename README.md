# pappelstrasse.ch

Website for the Pappelstrasse residential project in Zurich.

## Tech Stack

- **Laravel** (PHP)
- **Tailwind CSS v4**
- **Vite**
- **Blade** components
- **Swiper** (gallery)
- **Flatfox** API integration (apartment listings)

## Pages

- Landing
- Living
- Offer
- Apartments
- Location
- Imprint / Privacy

## Development

```bash
# Install dependencies
composer install
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
